CSS Display 속성 비교

모두 똑같은 <div> 태그이지만, CSS display 속성에 따라 배치가 달라집니다.

1. Block (블록 요소)

특징: 혼자서 한 줄을 다 차지합니다. (줄바꿈 발생)

블록 박스 1 (div 기본값)
블록 박스 2 (너비가 100%입니다)
블록 박스 3 (너비를 줄여도 줄바꿈 됨)

2. Inline (인라인 요소)

특징: 글자처럼 옆으로 나란히 배치됩니다. 내용만큼만 크기를 가집니다.

인라인 박스 1
인라인 박스 2
인라인 박스 3 (내용이 길어지면 박스도 커짐)


주의: 인라인 박스는 CSS에서 width: 300px를 줬음에도 불구하고 사이즈가 적용되지 않고 무시됩니다.
